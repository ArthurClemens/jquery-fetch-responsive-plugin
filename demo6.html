Demo 6: Google Static Map image

<img class="my-image" data-range="320-640" data-step-size="40" data-ratio="2.5" data-high-resolution="auto" />

This demo creates dynamic calls to Static Maps API V2. For non-business use, Google limits the width to 640px wide (either normal or high resolution), so we use range to 640 and set highResolution to "auto".
